# ICP Роли И Сценарии

## Роли для проверки

### 1. Собственник / CEO SMB

Смотрит на:

- скорость управленческой отчетности;
- понятность цифр;
- качество решений;
- скорость подготовки КП.

### 2. Коммерческий директор / sales lead

Смотрит на:

- подготовку КП;
- согласование условий;
- контроль качества материалов;
- скорость ответа клиенту.

### 3. Операционный руководитель / finance / controller

Смотрит на:

- регулярную отчетность;
- консолидацию данных;
- ручную сборку таблиц;
- надежность цифр.

### 4. Аналитик / power user Excel

Смотрит на:

- сложные файлы;
- множественные листы;
- интерпретацию данных;
- подготовку выводов для других.

## Сценарии для валидации

### Сценарий 1. Управленческая отчетность

- собрать данные;
- свести их;
- интерпретировать;
- выдать понятный отчет руководителю.

### Сценарий 2. КП и согласование

- собрать вводные;
- подготовить КП;
- согласовать;
- отправить клиенту без задержек и ошибок.

### Сценарий 3. Сложный Excel

- открыть тяжелую выгрузку;
- понять, что в ней важно;
- вытащить инсайты;
- подготовить результат для принятия решения.
