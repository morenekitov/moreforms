# Текущий Workflow И User Journey

## Flow 1. Управленческая отчетность сегодня

1. Собрать данные из разных источников.
2. Ручками свести их в Excel.
3. Проверить корректность цифр.
4. Подготовить версию для руководителя.
5. Переделать после уточнений.

Где боль:

- долго;
- много ручных шагов;
- ошибки трудно отследить;
- итог часто не объясняет, что делать дальше.

## Flow 2. КП и согласование сегодня

1. Получить вводные.
2. Собрать КП из старых шаблонов и файлов.
3. Согласовать внутри.
4. Пересобрать после правок.
5. Отправить клиенту.

Где боль:

- версионность;
- долгие согласования;
- зависимость от конкретного человека;
- сложно быстро адаптировать КП под новый кейс.

## Flow 3. Сложный Excel сегодня

1. Получить выгрузку.
2. Понять структуру файла.
3. Очистить и привести данные.
4. Попробовать найти закономерности.
5. Собрать выводы для бизнеса.

Где боль:

- трудно быстро понять файл;
- много ручной интерпретации;
- инсайты неочевидны;
- есть разрыв между данными и решением.
